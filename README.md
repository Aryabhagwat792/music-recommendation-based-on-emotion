# music-recommendation-based-on-emotion
Emotion-Based Music Recommender is a Python app that suggests songs based on user emotion. It uses a simple emotion detection/selection system and a songs dataset to match moods like happy, sad, or angry. Built with Python, Tkinter, and CSV, it provides a basic GUI for mood-based music recommendations.
Enhanced Emotion Music Recommendation App 
Overview 
The Enhanced Emotion Music App is a Python-based desktop application that 
recommends songs based on user emotions. It uses a Tkinter GUI and is 
designed using a modular structure for better readability maintainability, and 
scalability. 
The system allows users to select emotions, search songs, and view previous 
search history. 
Features 
●  Emotion-based song recommendation 
●  Music suggestions for different moods 
●  Song search functionality 
●  User search historytracking 
●  Clean and interactive Tkinter GUI 
●  Lightweight and fast execution 
●  Modular architecture (separate files for logic, UI, and history)
Supported Emotions 
happy – energetic and uplifting songs   
sad –emotional and calm songs   
angry –powerful and motivational songs   
Project Structure 
Technologies Usedd 
Python 3 
Tkinter (GUI framework) 
File handling (for history storage) 
Basic Python logic (rule-based system)
